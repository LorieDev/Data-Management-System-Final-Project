QuickTracker is a Java-based Database Management System (DMS) that allows users to manage inventory items using a graphical interface (GUI) and a MySQL backend.
The system supports full CRUD operations (Create, Read, Update, Delete), input validation, and a custom feature that calculates the total value of all inventory.
